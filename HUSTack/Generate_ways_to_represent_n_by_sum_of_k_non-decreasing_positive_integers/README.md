# Description

Given two positive integers k and n. Write a program to generate(x1, x2, ..., xk) such that x1 <= x2 <= . . . <= xk and x1 + x2 + . . . + xk = n.

Input

- Line 1 contains: k and n (1 <= k <= n <= 40)

Output

- Each line contains: x1, x2, . . ., xk separated by a SPACE character

Example

Input

3  7

Output

1 1 5

1 2 4

1 3 3

2 2 3
