# Description

Cho hai số nguyên dương a và b rất lớn (số chữ số có thể lên tới 10000). Hãy tính và in ra tích của 2 số đó.

Input

- Dòng 1: ghi số a

- Dòng 2: ghi số b

Output

Ghi kết quả a*b

Example

Input

123

654

Output

80442

Example

Input

100000000000000000

123456789

Output

12345678900000000000000000
